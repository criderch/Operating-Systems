Operating Systems provide the functions for your computer. 
Pushing the button to start your computer is one example of an operating system function
If you need to restart your computer it is the operating system that is taking care of that.
Each operating system has a different method to restart your computer.
Another operating system function is shutting down your computer.
There are several ways a person may choose to shut down their computer.
These options would be to completely power off or place into a sleep mode.
A third operating system function is providing a user interface.
A user interface is the control of how a person interacts with their computer and how that information is displayed
Examples of this would be a graphical interface, which is the visual images, and the command line interface such as a computers function keys.
